## Color Coding System

All images follow consistent visual identity:

- **Gold (#FFD000):** Bitcoin, positive outcomes, hard money, freedom
- **Red (#FF4444):** Fiat, inflation, warnings, centralized control
- **Blue (#4477FF):** Data, comparisons, traditional finance, neutral
- **Gray (#808080, #C0C0C0, #2A2A2A):** Context, neutral elements, supporting data
- **Green (#10B981):** Success, positive outcomes, security
- **Orange (#FFA500):** Transitions, moderate risk, emerging trends
- **Black (#000000):** Universal background for maximum contrast

## Quality Assurance Checklist

Before finalizing any image, verify:
- [ ] Matches visual identity (black background, correct colors)
- [ ] 16:9 aspect ratio (1920×1080px)
- [ ] Text is legible at various screen sizes
- [ ] No photorealistic elements (unless specified)
- [ ] Educational tone (not propaganda or hype)
- [ ] Optimized file size (<250KB for most, <300KB maximum)
- [ ] WebP format for performance, PNG as fallback
- [ ] Alt text prepared for accessibility
